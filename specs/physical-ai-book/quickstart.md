# Quickstart Guide: Setting up the Book Project

This guide provides the steps to set up the Docusaurus project for the "Physical AI and Humanoid Robotics Book".

## Prerequisites

-   [Node.js](https://nodejs.org/en/) version 20.x or higher.
-   [Yarn](https://yarnpkg.com/) (or npm).

## 1. Create the Docusaurus Site

First, create a new Docusaurus site with the classic template.

```bash
npx create-docusaurus@latest physical-ai-book classic
```

This will create a new directory named `physical-ai-book` with a sample Docusaurus site.

## 2. Project Structure

Navigate into the new directory:
```bash
cd physical-ai-book
```

The project structure will be as follows:

```
physical-ai-book/
├── docs/
├── src/
│   ├── components/
│   └── css/
├── static/
│   └── img/
├── docusaurus.config.js
├── package.json
└── sidebars.js
```

## 3. Configure `docusaurus.config.js`

Open `docusaurus.config.js` and customize the site's title, tagline, and other metadata.

```javascript
const config = {
  title: 'Physical AI and Humanoid Robotics',
  tagline: 'A Hands-On Guide to Building Intelligent Robots',
  // ... other configurations
};
```

## 4. Set up `sidebars.js`

Modify `sidebars.js` to create a custom sidebar for the book's content.

```javascript
const sidebars = {
  bookSidebar: [
    {
      type: 'autogenerated',
      dirName: '.', // or specify your directory structure
    },
  ],
};
```

## 5. Start the Development Server

Run the following command to start the development server:

```bash
yarn start
```

This will open a new browser window with the Docusaurus site running at `http://localhost:3000`.

## 6. Content Development

You can now start creating the book's content by adding Markdown files to the `docs` directory, following the structure outlined in the `plan.md` file.
